# richesse.club Designer Handoff Brief

이 문서는 richesse.club 전용 디자이너 브리프입니다.
디자인 기준은 같은 폴더의 `BRAND_GUIDE.md` 를 우선하며, 핵심 방향은 `Swiss design`, `텍스트+사진/영상 스틸만`, `Pretendard 고정`, `모노크롬`, `커버와 본문 슬라이드 분리`, `거의 고정된 본문 템플릿`입니다.

## 4) 비주얼 디렉션
```markdown
- 톤 키워드: swiss / refined / restrained / editorial
- 금지 요소: 아이콘 중심 구성, 일러스트, 과한 그라데이션, 글로우, 블러, 주황색 중심 강조
- 컬러 기준: 흑백/회색 중심, accent 없이 진행
- 강조 전략: 크기, 정렬, 여백, 폰트 두께 차이만으로 해결
- 사진 원칙: 사진이나 영상 스틸은 사용할 수 있지만 텍스트보다 앞서면 안 됨
- 서체 원칙: Pretendard만 사용, body 슬라이드 헤드라인은 기본 `700` weight 사용
- 커버 원칙: Slide 1은 후킹을 위해 상대적으로 자유롭게 구성 가능
- 본문 원칙: Slide 2~N은 거의 항상 같은 에디토리얼 템플릿을 반복
- 본문 기본 구조: 상단 사진 블록 -> 번호 마커([02]) + 가는 구분선 -> 헤드라인 -> 충분한 분량의 본문 카피
- 번호 원칙: 번호 마커는 너무 작게 죽이지 말고 `24px ~ 30px` 범위에서 구조 anchor 역할을 하게 둘 것
- 타이포 원칙: 헤드라인 자간은 약간 조이고, body 카피는 자간과 행간을 함께 다듬어 답답하지 않게 읽히게 할 것
- 본문 카피 원칙: body 슬라이드는 텍스트 밀도가 살아 있어야 하며, 짧은 한두 줄로 끝내지 않음
- 정렬 원칙: 번호, 선, 헤드라인, 본문 시작선은 같은 축으로 정리
- 이미지 운용 원칙: 원본 비율이 달라도 상단 사진 블록 높이는 유지하고, 슬라이드마다 crop/focal point를 직접 조정
``` 

## 5) 슬라이드별 메모
```markdown
- Slide 1:
  - 역할: Cover
  - 기획 의도: 피드에서 시선을 멈추게 하고 주제를 한 번에 인지시킴
  - 필수 요소: 제목, 주제 인상, 브랜드 톤 유지

- Slide 2:
  - 역할: First Body
  - 기획 의도: 커버 이후 본문 리듬을 확정하는 첫 기준 슬라이드
  - 필수 요소: 상단 사진 블록, 번호 마커, 구분선, 강한 헤드라인, 충분한 분량의 본문

- Slide 3~N-1:
  - 역할: Body
  - 기획 의도: 같은 구조 안에서 메시지 1개씩 차분하게 전개
  - 필수 요소: Slide 2와 동일한 골격 유지, 메시지 1개, 사진 톤 일관성, 충분한 본문 밀도

- Final:
  - 역할: Closing / CTA
  - 기획 의도: 전체 메시지를 정리하고 마지막 행동을 안내
  - 필수 요소: 마무리 문장 또는 CTA, 앞 슬라이드와 같은 톤 유지 여부 확인
```

## 6) 본문 슬라이드 제작 규칙
- 사진은 본문 설명을 위한 분위기 보조 역할로 사용하고, 텍스트보다 먼저 튀지 않게 한다.
- 사진은 상단 넓은 직사각형 1개를 기본으로 하고, body 슬라이드에서는 풀블리드보다 inset 구성을 우선한다.
- 번호 마커는 구조가 읽힐 만큼 충분히 크게 두며, 구분선은 시각 장식이 아니라 레이아웃 기준선 역할을 해야 한다.
- 헤드라인은 한 줄 또는 두 줄 안에서 끝내는 것을 우선하고, weight는 `700`을 기본으로 잡는다. 자간은 다소 타이트하게 조정한다.
- 본문은 메시지를 충분히 설명할 수 있을 정도의 분량을 확보하고, 행간과 자간을 같이 보정해 읽는 속도가 끊기지 않게 한다.
- 본문은 보통 4~7행 안팎의 밀도 있는 문단으로 보고, 핵심 주장만 던지고 끝내지 않는다. 이유, 맥락, 해석 중 최소 하나는 살아 있어야 한다.
- 슬라이드가 여러 장 이어질수록 번호 위치, 선 길이, 텍스트 폭, 상하 간격이 흔들리지 않아야 한다.
- 사진은 원본마다 비율과 피사체 위치가 다르므로 한 값으로 고정하지 않는다. 얼굴, 손, 중심 오브젝트가 잘리지 않도록 crop과 focal point를 슬라이드마다 직접 점검한다.
- 사진 위에 본문을 얹거나, 각 슬라이드마다 전혀 다른 템플릿으로 바꾸는 방식은 richesse.club 본문 흐름과 맞지 않는다.
- 텍스트가 부족하면 디자인 요소를 늘리는 대신 원고를 보강하는 쪽으로 판단한다.

## 7) 확인 메모
- 브랜드 가이드의 Swiss design 원칙을 지켰는가?
- 텍스트와 사진/영상 스틸 외 요소를 과하게 쓰지 않았는가?
- Pretendard 외 서체를 사용하지 않았는가?
- 지나치게 두꺼운 타이포와 주황 accent가 들어가지 않았는가?
- 공통 가독성 기준과 4:5 규격은 지켰는가?
- Cover를 제외한 본문 슬라이드가 같은 구조 안에서 안정적으로 반복되는가?
- 번호 마커, 구분선, 헤드라인, 본문 축이 슬라이드마다 어긋나지 않는가?
