# richesse.club Brand Guide

이 문서는 richesse.club의 주제 선정과 콘텐츠 패키징 기준이다. 브랜드를 너무 좁게 정의하지 않고, wealth를 넓게 다루되 어떤 방식으로 분류하고 좁혀야 하는지에 집중한다.

## 1) 브랜드 정의

- richesse.club은 `부를 만드는 사람들의 생각, 판단, 구조를 다루는 비즈니스 에디토리얼 미디어`다.
- 핵심은 **Business**다. 창업, 전략, 성장, 돈의 구조를 먼저 다룬다.
- 실리콘밸리 창업자·VC·사상가의 인터뷰와 인사이트를 한국 독자가 저장하고 싶은 형태로 패키징한다.
- Places, Taste, Culture는 브랜드가 자리를 잡은 후 조금씩 확장한다. 지금은 Business와 People이 중심이다.
- 좋은 richesse 콘텐츠는 `강한 후킹`과 `실제로 쓸 수 있는 인사이트`가 같이 살아 있다.

## 2) 기본 방향

- 카테고리는 넓게 잡는다.
- 실제 포스트를 만들 때는 주제와 포맷을 좁힌다.
- 모든 콘텐츠를 같은 문법으로 만들지 않는다.
- 주제마다 맞는 포맷을 쓴다.
- 모든 포스트가 깊을 필요는 없지만, 모두 wealth라는 큰 세계 안에는 있어야 한다.

## 3) 브랜드 톤

- 고급스럽지만 숨막히지 않아야 한다.
- 세련됐지만 가르치려 들거나 아는 척하지 않아야 한다.
- 저장하고 싶지만 피곤하지 않아야 한다.
- 과장된 허세나 가르치는 듯한 무거운 톤('~해야 한다')은 피한다.
- **표지/헤드라인/리스트 문구:** 짧고 강하게 끝맺는다. 여기서는 힘 있는 문어체('~한다', '~하는가' 등)를 써도 무방하다.
- **글(단락) 형식의 본문과 캡션:** 기본은 차분하고 절제된 문어체(`~다`, `~한다`)를 사용한다.

## 4) 콘텐츠를 보는 4개의 축

richesse.club은 주제를 고를 때 아래 네 축으로 동시에 본다.

### Category

무엇을 다루는가.

### Format

어떤 방식으로 보여주는가.

### User Value

사람들이 왜 저장하거나 공유하는가.

### Depth

얼마나 가볍게 혹은 깊게 다루는가.

이 네 축을 같이 봐야 richesse가 유연해진다.

## 5) Category


### Business

- 창업가
- 브랜드
- 회사
- 일하는 방식
- 대표의 기준
- 가격 전략
- 사업 감각

### Money

- 자산
- 현금흐름
- 소비
- 가격
- 레버리지
- 소유권
- 돈이 쓰이는 방식
- 부의 구조

### People

- 부자
- 창업가
- 브랜드를 만든 사람
- 잘 사는 사람들의 선택
- 인물 관찰

### Culture

- 사교
- 매너
- 취향의 신호
- 멤버십 문화
- 좋은 서비스
- 계층감
- 라이프스타일 코드

## 6) Format

카테고리보다 더 중요한 것이 포맷이다. 같은 카테고리라도 포맷이 달라지면 완전히 다른 콘텐츠가 된다.

### 큐레이션형

- 여러 장소, 제품, 브랜드를 묶어서 추천한다.
- 예: `서울에서 분위기 좋은 라운지 5곳`


### 정리형

- 헷갈리는 개념, 직책, 구조를 한 번에 정리한다.
- 예: `C레벨 직책 6개 정리`

### 비교형

- 비슷한 대상의 차이를 보여준다.
- 예: `호텔 라운지와 private club은 무엇이 다를까`


### 리스트형

- 저장성이 높은 정보나 추천을 짧고 명확하게 나열한다.
- 예: `중요한 미팅 때 가기 좋은 레스토랑 7곳`

### 케이스형

- 한 사람, 브랜드, 회사, 공간을 사례로 풀어낸다.
- 예: `이 브랜드가 비싸도 팔리는 이유`

### 인사이트형

- 구조, 기준, 판단을 해석한다.
- 예: `좋은 대표는 왜 가격 결정권을 놓치지 않을까`

### 해설형

- 문화나 신호, 장면의 의미를 읽어준다.
- 예: `왜 어떤 공간은 들어가자마자 조용한 권위가 느껴질까`

## 7) User Value

같은 주제라도 사람들이 보는 이유는 다르다. 이걸 구분해야 한다.

### 저장형

- 나중에 다시 보고 싶어서 저장한다.
- 정리형, 리스트형, 입문형에서 강하다.

### 공유형

- 친구나 동료에게 보내고 싶어서 공유한다.
- 비교형, 정리형, 가벼운 인사이트형에서 강하다.

### 공감형

- `맞아, 나도 그렇게 느꼈어`라는 반응을 만든다.
- Culture, People, 해설형에서 강하다.

### 정보형

- 바로 도움이 되거나 유용해서 본다.
- 큐레이션형, 소개형, 입문형에서 강하다.

### 브랜딩형

- richesse.club이 어떤 감각과 시선을 가진 계정인지 보여준다.
- 케이스형, 인사이트형, 해설형에서 강하다.

### User Value 규칙

- 한 포스트에는 `메인 가치 1개`를 먼저 정한다.
- 필요하면 `서브 가치 1개`까지만 추가한다.
- 세 개 이상 붙이기 시작하면 포스트의 쓰임새가 흐려진다.
- 메인 가치가 저장형이면 제목과 구성도 저장하기 쉽게 간다.
- 메인 가치가 브랜딩형이면 정보량보다 richesse.club다운 시선이 더 중요하다.

## 8) Depth

### Light

- 부담 없이 본다.
- 추천, 소개, 간단 정리, 입문형에 잘 맞는다.

### Medium

- 가볍게 들어오지만 읽고 나면 생각할 거리가 남는다.
- richesse.club의 기본값으로 가장 많이 쓴다.

### Deep

- 구조와 기준을 강하게 읽어낸다.
- 너무 자주 쓰면 피곤해진다.
- 계정의 깊이를 보여줄 때 선택적으로 쓴다.

## 9) Timing

좋은 주제라도 언제 올리느냐에 따라 운영 가치가 달라진다.

### Now

- 지금 바로 올릴 이유가 있는 주제다.
- 시즌, 이슈, 날씨, 특정 시기, 최근 대화 흐름과 연결된다.

### Evergreen

- 언제 올려도 괜찮은 상시 저장형 주제다.
- richesse.club 운영의 기본 재고가 되는 주제다.

### Seasonal

- 특정 계절, 휴가철, 연말, 연초, 행사 시즌에 특히 잘 맞는다.
- 평소에 보관해두고 타이밍에 맞춰 꺼내 쓰는 주제다.

## 10) 실전 분류 예시

### 예시 1

- Topic: `C레벨 직책 6개 정리`
- Category: `Business`
- Format: `정리형`
- User Value: `저장형`
- Depth: `Light`
- Timing: `Evergreen`

### 예시 2

- Topic: `서울에서 대화하기 좋은 라운지 5곳`
- Category: `Places`
- Format: `큐레이션형`
- User Value: `저장형 + 정보형`
- Depth: `Light`
- Timing: `Now` 또는 `Seasonal`

### 예시 3

- Topic: `좋은 대표는 왜 가격 결정권을 놓치지 않을까`
- Category: `Business + Money`
- Format: `인사이트형`
- User Value: `브랜딩형 + 공유형`
- Depth: `Deep`
- Timing: `Evergreen`

### 예시 4

- Topic: `와인을 처음 마실 때 보기 좋은 기준`
- Category: `Taste`
- Format: `입문형`
- User Value: `저장형 + 정보형`
- Depth: `Light`
- Timing: `Evergreen`

### 예시 5

- Topic: `왜 어떤 공간은 서비스의 리듬부터 다를까`
- Category: `Places + Culture`
- Format: `해설형`
- User Value: `공감형 + 브랜딩형`
- Depth: `Medium`
- Timing: `Evergreen`

## 11) 주제 선정 규칙

- 먼저 카테고리를 넓게 본다.
- 그다음 포맷을 정한다.
- 그다음 사용자 가치가 무엇인지 정한다.
- 그다음 깊이를 정한다.
- 마지막으로 timing을 정한다.
- 이 다섯 가지가 정리되면 같은 주제라도 richesse다운 패키징이 가능해진다.

주제를 고를 때는 아래 질문을 본다.

- 이건 어느 category에 들어가는가?
- 이건 어떤 format이 가장 자연스러운가?
- 사람들은 왜 이걸 저장하거나 공유할까?
- 메인 user value는 무엇이고, 서브는 필요한가?
- 이건 light, medium, deep 중 어디가 맞는가?
- 지금 올릴 주제인가, 상시 재고인가, 시즌성 주제인가?
- richesse.club의 세계 안에 들어오는가?

## 12) richesse.club답지 않은 것

- 뉴스 요약만 있고 richesse.club 패키징이 없는 콘텐츠와 유사하게, 구조 없이 나열만 하는 팁 리스트
- 허세 섞인 부자 콘텐츠 (명품 나열, 비싼 것 자랑)
- 뉴스 요약만 있고 richesse.club 패키징이 없는 콘텐츠
- 제목은 자극적이지만 내용이 없는 콘텐츠
- 소스에 없는 내용을 창작해서 붙인 콘텐츠
- 럭셔리 라이프스타일 추천 (아직 이 단계가 아님)
- category는 맞지만 format이 어색한 콘텐츠

## 14) 운영 메모

- richesse.club은 `카테고리는 넓게, 포스트는 좁게`가 기본 원칙이다.
- Places와 Taste는 큐레이션형, 소개형, 입문형이 자연스럽다.
- Business와 Money는 정리형, 케이스형, 인사이트형이 자연스럽다.
- People과 Culture는 해설형, 비교형, 케이스형이 자연스럽다.
- 한 포스트를 기획할 때는 반드시 `Category / Format / User Value / Depth / Timing`을 함께 적는다.
- 운영할 때는 `Now`와 `Evergreen`을 섞고, `Seasonal`은 미리 쌓아두는 방식이 좋다.
