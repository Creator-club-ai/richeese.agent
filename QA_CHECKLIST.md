# 게시 전 QA 체크리스트

게시 직전, 아래 항목을 모두 체크해야 합니다.

## 1) 사실/메시지 정확성
- [ ] 핵심 주장에 근거가 있다 (출처 또는 검증 완료)
- [ ] 콘텐츠 유형별 최소 출처 수/신뢰도 티어 기준을 충족했다
- [ ] 과장/오해 소지가 있는 표현이 없다
- [ ] 제목과 본문 메시지가 일치한다

## 2) 톤앤매너/브랜드 일관성
- [ ] 타겟(2030 예비/초기 창업가)에 맞는 어조다
- [ ] 지나치게 공격적/권위적인 표현이 없다
- [ ] 브랜드 컬러/스타일 가이드와 충돌이 없다

## 3) 가독성/디자인 품질
- [ ] 슬라이드당 텍스트 밀도가 과하지 않다
- [ ] 슬라이드 수가 기본 6~10장 범위이며, 11~12장은 확장 사유가 명확하다
- [ ] 강조 포인트가 명확하다 (`**강조**`)
- [ ] 오탈자/띄어쓰기 오류가 없다
- [ ] 모바일 화면에서 핵심 문장이 잘린 부분이 없다

## 4) 퍼포먼스 요소
- [ ] 첫 슬라이드 또는 첫 3초 훅이 강하다
- [ ] CTA가 명확하다 (저장/공유/댓글 유도)
- [ ] 해시태그가 주제와 타겟에 맞는다

## 5) 운영/배포 준비
- [ ] 파일명 규칙이 일관적이다
- [ ] 게시 일정/채널이 캘린더와 일치한다
- [ ] 승인자(A)가 최종 확인했다
