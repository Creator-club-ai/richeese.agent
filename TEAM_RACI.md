# 콘텐츠팀 RACI 매트릭스

아래 표의 `이름/직책`은 실제 팀 상황에 맞게 교체하세요.

## 역할 정의
- R (Responsible): 실무 수행 책임자
- A (Accountable): 최종 의사결정/승인자
- C (Consulted): 사전 협의 대상
- I (Informed): 결과 공유 대상

## 기본 역할
- `Content Lead` (팀 리드)
- `Researcher` (리서치)
- `Planner` (기획)
- `Editor` (원고)
- `Designer` (카드뉴스 디자인)
- `Reels PD` (영상)
- `Publisher` (업로드/배포)
- `Analyst` (성과 분석)

## RACI 표
| 업무 단계 | Content Lead | Researcher | Planner | Editor | Designer | Reels PD | Publisher | Analyst |
|---|---|---|---|---|---|---|---|---|
| 월간 주제 선정 | A | C | R | C | I | I | I | C |
| 주간 콘텐츠 캘린더 확정 | A | C | R | C | C | C | I | C |
| 리서치 브리프 작성 | A | R | C | C | I | C | I | I |
| 카드뉴스 기획안 작성 | I | C | R | C | C | I | I | I |
| 카드뉴스 원고 작성 | I | C | C | R | C | I | I | I |
| 카드뉴스 디자인/최종 산출 | I | I | I | C | R | I | I | I |
| 릴스 대본/자막 구성 | I | C | C | R | I | C | I | I |
| 릴스 렌더링 | I | I | I | C | I | R | I | I |
| 전달 전 QA 검수 | A | C | C | R | C | C | C | I |
| 산출물 전달 (업로드는 오너) | A | I | I | I | I | I | R | I |
| 성과 집계/리포트 | I | C | I | I | I | I | C | R |
| 월간 회고 및 개선안 확정 | A | C | C | C | C | C | C | R |

## 운영 규칙
- 한 업무에 A는 반드시 1명만 둡니다.
- R은 복수 가능하지만, 최종 제출자는 1명으로 지정합니다.
- 게시 전 QA 미통과 시 Publisher는 게시를 보류합니다.
