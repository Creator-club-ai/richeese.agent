# Editorial PDCA Review Rubric

Use this rubric before routing to the next stage.

## Planner Gate

Pass when all are true:

- the direction is singular, not bundled
- `Category / Format / User Value / Depth / Timing` are explicit
- there is a clear save reason
- the angle is source-backed
- the structure is writable without re-planning

Common failure tags:

- `too-broad`
- `too-generic`
- `weak-save-value`
- `weak-source`
- `format-mismatch`
- `off-brand`

## Editor Gate

Pass when all are true:

- Slide 1 hook is sharp, concrete, and compressed enough for a cover
- each slide advances the logic instead of paraphrasing the previous slide
- body copy is compressed and readable on carousel canvas
- the tone matches richesse.club
- strong claims are still evidence-safe

Common failure tags:

- `hook-weak`
- `cover-overexplained`
- `body-flat`
- `too-wordy`
- `self-help-drift`
- `evidence-thin`
- `repetitive-structure`

## Designer Gate

Pass when all are true:

- layout rhythm matches the content density
- the visual direction supports the angle instead of decorating it
- hierarchy is obvious on `1080 x 1440`
- design notes are production-ready, not abstract mood words
- the result avoids generic luxury cliches

Common failure tags:

- `visual-mismatch`
- `hierarchy-weak`
- `pacing-issue`
- `generic-luxury`
- `type-led-mismatch`

## Route Table

- `weak-source`, `evidence-thin`, `fact-risk-critical` -> `research-desk`
- `too-broad`, `too-generic`, `weak-save-value`, `format-mismatch` -> `content-planner`
- `hook-weak`, `cover-overexplained`, `body-flat`, `too-wordy`, `repetitive-structure` -> `editor`
- `visual-mismatch`, `hierarchy-weak`, `pacing-issue`, `generic-luxury` -> `designer`

## Adversarial Pass

If the output looks too easily approved, run one extra adversarial review.

Trigger it when:

- the angle feels familiar but clean
- the copy is polished yet unsurprising
- the draft is structurally correct but may still be generic

Use the adversarial pass to answer:

- would this still feel save-worthy if the brand name were removed?
- is this truly richesse.club, or just competent business content?
