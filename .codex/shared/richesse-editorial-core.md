# Richesse Editorial Core

> 이 문서는 의도적으로 비어 있다.

브랜드 판단 기준과 전략은 active profile 안에서만 관리한다. Claude가 스킬을 실행할 때 읽어야 할 권위는 다음 문서이다:

- `brands/richesse-club/BRAND_GUIDE.md` — 5축(Category/Format/User Value/Depth/Timing), 톤, anti-patterns
- `brands/richesse-club/CONTENT_STRATEGY.md` — 믹스, 발행 페이스, 피해야 할 것
- `brands/richesse-club/RUNTIME_PROFILE.md` — 런타임 변수 (소스 경로 등)

`AGENTS.md`의 read order에서 이 파일을 그대로 두는 이유는 과거 참조(`.codex/shared/richesse-editorial-core.md`)를 끊지 않기 위함이다. 브랜드 내용을 여기에 다시 쓰지 말 것. 중복이 생기면 판단 근거가 두 군데에 흩어져 어느 한쪽이 반드시 낡는다.
