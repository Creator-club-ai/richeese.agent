# 에이전트 콜사인

운영 중 빠르게 역할을 호출하기 위한 콜사인 표입니다.

## 콜사인 매핑
- `리드` -> `dasarom4` (최종 우선순위/승인)
- `스카웃` -> `content_researcher` (소스 탐색, Research Brief)
- `플랜` -> `content_planner` (기획/구조화)
- `카피` -> `content_editor` (슬라이드 문안/캡션)
- `카드` -> `slide_designer` (디자이너 브리프/디자인 완성, 필요 시 초안 렌더)
- `모션` -> `remotion_pd` (릴스 제작)

## 빠른 호출 예시
1. "스카웃, [주제] Research Brief 만들어줘"
2. "플랜, 브리프로 8장 캐러셀 기획해줘"
3. "카피, 슬라이드별 원고 + 강조 후보 + 비주얼 힌트 작성해줘"
4. "카드, 브리프 기준으로 슬라이드 디자인 방향 정리해줘"
