# 디자이너 핸드오프 브리프 — 2026 창업패키지 일정표

콘텐츠 ID: 2026-gov-startup-packages-schedule-v1
포맷: Instagram Carousel (1080x1440)
작성일: 2026-03-06 (KST)

## 1) 콘텐츠 목표
- 핵심 목표(1문장): 2026년 예비·초기·도약 창업패키지의 접수기간을 한 장에 저장 가능한 형태로 정리한다.
- 타겟 독자: 예비창업자/초기창업자(지원사업을 "언제" 준비해야 하는지부터 알고 싶은 사람)
- 보고 나서 원하는 행동(CTA): 저장 + 동료에게 공유 + 제출 직전 원문 확인 습관화

## 2) 핵심 메시지 우선순위
1. 접수기간(날짜)만 빠르게 정리
2. 통합공고 vs 개별공고 구분(헷갈림 제거)
3. 변경 가능성 안내 + 원문 재확인 CTA

## 3) 비주얼 디렉션 (디자이너 재해석용)
- 톤: Swiss / editorial / grid-first / big type + generous whitespace
- 컬러: Black / White / Orange(Accent: #FF5100)만 사용 (중간 회색은 보조 텍스트에서만)
- 강조 전략: 날짜(기간) + 1~2개 핵심 단어만 오렌지/룰/하이라이트로 처리
- 그래픽 언어: 굵은 룰(line), 도트 패턴(희미), 라운드 캡슐(태그/기간)
- 금지: 과한 아이콘 세트, 장식용 일러스트 난사, 의미 없는 드롭섀도우/글로우
- 분위기 참조(Unsplash 키워드): "abstract grid", "paper texture", "modern office black" (필요 시)

## 4) 슬라이드별 의도 (역할 + 추천 시각 패턴)

- Slide 1
  - 역할: Cover
  - 추천 시각 패턴: Text-first (dark) + dot pattern
  - 필수 요소: 2026 / 일정표 / Save 유도
  - 기획 의도: 피드에서 멈추는 강한 타이포 임팩트

- Slide 2
  - 역할: Context
  - 추천 시각 패턴: Text-first + 2-card 대비
  - 필수 요소: 통합공고 vs 개별공고 구분
  - 기획 의도: "내가 헷갈렸던 포인트"를 3초 내 해소

- Slide 3
  - 역할: Body
  - 추천 시각 패턴: Data/List (timeline)
  - 필수 요소: 1~3월 흐름, 딥테크(참고) 포함
  - 기획 의도: 전체 흐름을 한 번에 감 잡게

- Slide 4
  - 역할: Body
  - 추천 시각 패턴: Text-first (big date)
  - 필수 요소: 초기(일반형) 접수기간
  - 기획 의도: 스크린샷/저장해도 읽히는 큰 날짜

- Slide 5
  - 역할: Body
  - 추천 시각 패턴: Text-first (big date)
  - 필수 요소: 도약(일반형) 접수기간 + 유형별 상이 노트
  - 기획 의도: 오해 방지(유형에 따라 다름)

- Slide 6
  - 역할: Body
  - 추천 시각 패턴: Text-first (big date) + 작은 배지("수정 공고")
  - 필수 요소: 예비(수정 공고) 접수기간
  - 기획 의도: 최신 공고 기준임을 명확히

- Slide 7
  - 역할: Summary
  - 추천 시각 패턴: Data/List (table)
  - 필수 요소: 3대 패키지 요약표 + 딥테크 참고
  - 기획 의도: 저장가치 극대화(이 장만 봐도 된다)

- Slide 8
  - 역할: Body → CTA
  - 추천 시각 패턴: Card/List (dark) + numbered checklist
  - 필수 요소: 접수 전 행동 3개
  - 기획 의도: 날짜를 행동으로 연결

- Slide 9
  - 역할: CTA
  - 추천 시각 패턴: Text-first + source box
  - 필수 요소: 저장/원문 확인 CTA, 출처 ID(짧게)
  - 기획 의도: 신뢰 확보 + 면책(변경 가능)
