# Richesse.club 브랜드 가이드

모든 콘텐츠 제작 시 참조하는 브랜드 가이드라인입니다.

---

## 브랜드 아이덴티티

- **브랜드명**: Richesse.club (리셰스클럽)
- **슬로건**: "성장하는 기업가들의 이야기"
- **타겟**: 20~40대, 스타트업/창업에 관심 있는 예비 창업가, 초기 창업자, MBA/경영 전공자, 스타트업 대표
- **톤앤매너**: 진정성, 인사이트, 성장, 실용성

---

## 콘텐츠 주제

### 핵심 카테고리

| 카테고리 | 주제 | 예시 |
|----------|------|------|
| **기업가정신** | 창업 마인드, 도전, 실패와 성장 | "10번 실패한 창업가가 말하는 것" |
| **스타트업** | 초기 단계, 팀 빌딩, 제품-시장 적합성 | "시리즈A 투자 받기 전 해야 할 것" |
| **CEO 인터뷰** | 경영 철학, 의사결정, 리더십 | "유니콘 대표가 말하는 팀 문화" |
| **비즈니스 인사이트** | 전략, 성장, 시장 분석 | "넷플릭스는 왜 광고 모델을 선택했나" |

### 콘텐츠 톤

- **진정성**: 포장하지 않은 날것의 경험
- **실용성**: 당장 적용 가능한 실전 팁
- **성장**: 실패를 배움으로 전환
- **인사이트**: 표면이 아닌 본질

## 톤앤매너 가이드

### 기본 톤 선택

Richesse.club은 두 가지 톤을 상황에 맞게 사용합니다:

| 톤 | 사용처 | 특징 |
|----|--------|------|
| **~다/명령조** | 쇼츠, 인스타 (임팩트 중시) | 강단 있고 확신에 찬 느낌 |
| **~습니다체** | 블로그, 뉴스레터 | 부드럽고 신뢰감 있는 느낌 |

> 인스타그램 슬라이드는 **두 가지 톤 모두** 생성하여 테스트

### 글쓰기 규칙 (필수)

1. **설명하는 느낌 ❌**
   - 나쁜 예: "이것은 중요한 개념입니다."
   - 좋은 예: "성공한 창업가들은 이걸 먼저 배운다."

2. **명사로 끊기 ❌**
   - 나쁜 예: "오직 성장." / "이게 전부."
   - 좋은 예: "그들이 보는 건 오직 성장 지표뿐이다."

3. **리듬감 있게**
   - 긴 문장 → 짧은 문장 패턴
   - 예: "스타트업 초기엔 모든 게 불확실하다. 하지만 한 가지는 확실하다. 빠르게 실행하는 팀이 이긴다."

4. **40자 제한 없음**
   - 자연스럽게 끊기
   - 무리하게 줄이지 말 것

5. **쉬운 한국어**
   - 어려운 용어 지양
   - 필요하면 설명 추가

### ✅ 권장 표현

- **진정성**: "솔직히 말하면", "실패했지만", "배웠다"
- **구체성**: 숫자, 사례, 타임라인 포함
- **실용성**: "당장 해볼 것", "첫 번째 단계는"
- **성장**: "배웠다", "깨달았다", "다음엔"

### ❌ 피해야 할 표현

- 과도한 과장: "인생이 180도 바뀝니다!"
- 뻔한 클리셰: "성공의 비밀", "꿀팁 대방출"
- 부정적 과다 사용: "실패자들의 특징 10가지"
- 전문 용어 남발: "MRR", "CAC" 등 설명 없이 사용

### 예시

```
❌ 나쁜 예
"이것만 하면 당신의 스타트업이 완전히 달라집니다!!!"
"성공의 비밀을 대공개합니다!"

✅ 좋은 예
"3번 실패한 창업가가 4번째에 성공한 이유"
"시리즈A 투자 받기 전, 이 지표는 꼭 준비하라"
"배민 김봉진 대표가 초기에 했던 결정"
```

## 콘텐츠 제작 원칙

### 1. 진정성 (Authenticity)
- 포장하지 않은 날것의 경험
- 실패 스토리도 솔직하게
- 과장하지 않는 팩트

### 2. 인사이트 (Insight)
- 표면이 아닌 본질
- "왜"에 집중
- 생각할 거리 제공

### 3. 실용성 (Actionable)
- 당장 적용 가능한 팁
- 구체적인 단계
- 실전 사례

### 4. 성장 (Growth)
- 실패 → 배움 → 성장
- 과정을 중시
- 다음 단계 제시
