# Content Analyzer (콘텐츠 분석기)

> **목적**: 다양한 소스에서 **텍스트 인사이트**를 추출하여 Instagram/Threads 콘텐츠 생성에 활용
> **사용처**: `/instagram-slides`, `/threads` 워크플로우

---

## 1. 입력 소스 유형

| 유형 | 예시 | 처리 방법 |
|------|------|----------|
| **유튜브 전사** | `.txt`, `.srt` | 자막 파일 직접 읽기 |
| **웹 아티클** | 블로그, 뉴스 | `read_url_content` 도구 |
| **PDF/문서** | 책 요약, 보고서 | PyPDF2 또는 직접 읽기 |
| **수동 입력** | 메모, 스크립트 | 직접 제공 |

---

## 2. 분석 프레임워크 (5단계)

### Step 1: 소스 읽기
원본 텍스트를 가져와 **구조화된 형태로 정리**

### Step 2: 핵심 메시지 추출
> "이 콘텐츠가 전달하려는 **단 하나의 핵심 메시지**는 무엇인가?"

- 한 문장으로 정의
- 다른 모든 내용은 이 메시지를 뒷받침하는 것

### Step 3: 인사이트 분류

| 분류 | 설명 | 예시 |
|------|------|------|
| **프레임워크** | 구조화된 개념/모델 | "유니콘의 4가지 조건" |
| **반전** | 일반적 믿음과 다른 주장 | "좋은 제품이 알아서 팔린다? 틀렸다" |
| **사례** | 구체적 예시/케이스 | "Facebook은 하버드에서 시작했다" |
| **명언** | 기억에 남는 한 줄 | "단순한 행동도 안 하면 복잡한 기능은 무의미" |
| **액션** | 실행 가능한 팁 | "프로덕트 훅과 유통만 먼저 확보하라" |

### Step 4: 콘텐츠별 가공

**Instagram용:**
```
- 시각적으로 표현 가능한 것 우선
- 숫자/리스트/대비 구조 선호
- 슬라이드당 1개 인사이트
```

**Threads용:**
```
- 대화/토론 유발 가능한 것 우선
- 의견이 갈릴 수 있는 주장
- 짧고 강렬한 문장
```

### Step 5: 출력 포맷

```markdown
## 📌 핵심 메시지
[한 문장]

## 💡 주요 인사이트 (우선순위순)

### 1. [제목]
- 유형: [프레임워크/반전/사례/명언/액션]
- 내용: [2-3문장]
- Instagram 적합도: ⭐⭐⭐ (3점 만점)
- Threads 적합도: ⭐⭐ (3점 만점)

### 2. [제목]
...

## 🎯 권장 콘텐츠 구성
- Instagram: [추천 인사이트 번호]
- Threads: [추천 인사이트 번호]

## 📝 추천 Hook
1. [후크 문장 1]
2. [후크 문장 2]
```

---

## 3. 분석 품질 체크리스트

- [ ] 핵심 메시지가 한 문장으로 명확한가?
- [ ] 인사이트가 원본의 주장을 왜곡하지 않는가?
- [ ] 각 인사이트가 독립적으로 이해 가능한가?
- [ ] 플랫폼별 적합도가 구분되어 있는가?
- [ ] Hook이 호기심을 유발하는가?

---

## 4. 예시

### 입력: David Sacks 인터뷰 (유니콘 기업의 4가지 조건)

### 출력:
```markdown
## 📌 핵심 메시지
유니콘 기업은 운이 아니라 **4가지 조건(시장, 훅, 유통, 방어기제)의 설계**로 만들어진다.

## 💡 주요 인사이트

### 1. 작은 독점에서 시작하라
- 유형: 반전
- 내용: 큰 시장을 처음부터 노리면 경쟁에 눌린다. Facebook도 하버드 하나에서 시작했다.
- Instagram: ⭐⭐⭐
- Threads: ⭐⭐

### 2. 기능이 아니라 행동을 만들어라
- 유형: 프레임워크
- 내용: Twitter는 글쓰기, Uber는 버튼 누르기. 단순한 반복 행동이 제품의 본질.
- Instagram: ⭐⭐⭐
- Threads: ⭐⭐⭐

## 📝 추천 Hook
1. "좋은 제품이면 알아서 팔린다고? 거의 항상 틀렸다."
2. "유니콘은 만들어지는 거다. 우연이 아니라."
```

## 5. 워크플로우 연결

```
소스 입력
    ↓
Content Analyzer (이 가이드)
    ↓
┌───────────────┬────────────────┐
│               │                │
↓               ↓                ↓
/instagram   /threads      (다른 용도)
```
