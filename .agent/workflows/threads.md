# /threads 워크플로우

유튜브 영상의 핵심 인사이트를 쓰레드 게시물로 변환하는 에이전트입니다.

## 사용법

```
# 유튜브 영상
/threads https://youtube.com/watch?v=영상ID

# 전사 파일 / 스크립트
/threads path/to/transcript.txt

# 웹 아티클
/threads https://blog.example.com/article
```

## 참조 문서

- `content_analyzer.md` - 다양한 입력 소스 처리 가이드
- `brand_guide.md` - Richesse 브랜드 가이드

---

## 실행 순서

### 1. 콘텐츠 추출

**유튜브 영상인 경우:**
// turbo
```powershell
python -m yt_dlp --no-check-certificate --skip-download --write-auto-sub --sub-lang "en,ko" --convert-subs srt -o "output/threads_source" "<YOUTUBE_URL>"
```

**전사 파일 / 아티클인 경우:**
파일을 직접 읽거나 read_url_content 도구 사용

### 2. 자막 분석 및 핵심 메시지 추출
`video_analyzer.md`를 참고하여 다음을 추출합니다:
- 핵심 인사이트 3~5개
- 인상적인 명언/인용구
- 실용적인 팁

### 3. 쓰레드 게시물 작성

**게시물 유형 선택:**

#### A. 단일 게시물 (짧은 인사이트)
```
[Hook - 강렬한 첫 문장]

[핵심 메시지 2~3줄]

[CTA - 팔로우/공유 유도]
```

#### B. 쓰레드 (연결 게시물 3~5개)
```
🧵 1/5
[주제 소개 + Hook]

🧵 2/5
[핵심 인사이트 ①]

🧵 3/5
[핵심 인사이트 ②]

🧵 4/5
[실행 팁 / 명언]

🧵 5/5
[요약 + CTA]
```

### 4. 결과물 저장
threads/ 폴더에 저장합니다.

## 쓰레드 작성 가이드

### 톤 선택

| 톤 | 특징 | 예시 |
|----|------|------|
| **~다/명령조** | 강렬하고 임팩트 있음 | "부자들은 시간을 산다." |
| **~습니다체** | 부드럽고 친근함 | "부자들은 시간을 삽니다." |

> **권장**: 쓰레드는 ~다/명령조가 더 효과적

### 글쓰기 규칙

1. **설명하는 느낌 ❌** - 팩트를 던지기
2. **명사로 끊기 ❌** - 완전한 문장으로
3. **리듬감** - 긴 문장 → 짧은 문장
4. **짧은 문장** - 한 문장 20자 이내 권장
5. **이모지 적절히** - 과하지 않게 (3~5개)

### 글자 수 제한
- **권장**: 300~500자
- **최대**: 500자 (쓰레드 제한)

### Hook 패턴 (첫 줄)

**효과적인 패턴:**
- 🔥 반전: "부자들은 절약하지 않습니다."
- 🔢 숫자: "30대에 은퇴한 사람들의 공통점 3가지"
- ❓ 질문: "왜 똑같이 일해도 월급만 다를까?"
- 💡 비밀: "억대 연봉자들이 절대 말 안 하는 것"

### 본문 스타일

1. **짧은 문장** - 한 문장 20자 이내
2. **줄바꿈 활용** - 2~3줄마다 빈 줄
3. **이모지 적절히** - 과하지 않게 (3~5개)
4. **구어체** - "~해요", "~합니다"

### CTA (Call to Action)

```
💾 저장해두고 리마인드 하세요
🔔 팔로우하면 매일 부의 인사이트
↩️ 공유해서 친구도 부자 마인드로
```

## 예시 게시물

### 단일 게시물 예시

```
부자들은 "돈을 아끼지" 않습니다.

대신 "시간을 법니다."

청소, 설거지, 운전...
이런 일에 시간 쓰는 대신
그 시간에 돈을 버는 일을 합니다.

월급쟁이: 시간 → 돈
부자: 돈 → 시간 → 더 많은 돈

이 사이클을 이해하면
돈에 대한 관점이 바뀝니다.

💾 저장해두세요

#부자마인드 #돈공부 #richesse
```

### 쓰레드 예시

```
🧵 1/4
"1조 자산가가 20대 때 했던 것"

대부분 20대에 뭘 해야 하는지 모릅니다.
하지만 이 분이 말한 3가지는 명확했습니다.

---

🧵 2/4
첫 번째: "모든 경험에 투자하라"

여행, 책, 사람...
20대에 쓴 경험비는
30대에 10배로 돌아옵니다.

---

🧵 3/4
두 번째: "실패를 수집하라"

20대 실패는
30대 성공의 데이터입니다.

---

🧵 4/4
세 번째: "건강에 복리를 적용하라"

20대 운동 습관 =
60대까지 건강 복리

💾 저장 + 🔔 팔로우

#richesse #부자마인드
```
