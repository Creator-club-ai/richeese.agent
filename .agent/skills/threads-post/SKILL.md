# Threads Post Generator

## 이 스킬은 언제 사용하나요?

- 사용자가 "쓰레드 포스트 만들어줘" 요청 시
- 사용자가 "Threads 게시물 써줘" 요청 시
- 사용자가 "쓰레드용 글 작성해줘" 요청 시

## 참조 문서

- **`.agent/config/content_analyzer.md`** - 텍스트 인사이트 추출
- `.agent/config/brand_guide.md` - Richesse 브랜드 가이드

---

## 실행 순서

### 1. 콘텐츠 추출

**유튜브 영상인 경우:**
```powershell
python -m yt_dlp --no-check-certificate --skip-download --write-auto-sub --sub-lang "en,ko" --convert-subs srt -o "output/threads_source" "<YOUTUBE_URL>"
```

**전사 파일 / 아티클인 경우:**
파일을 직접 읽거나 read_url_content 도구 사용

### 2. 자막 분석 및 핵심 메시지 추출
- 핵심 인사이트 3~5개
- 인상적인 명언/인용구
- 실용적인 팁

### 3. 쓰레드 게시물 작성

**게시물 유형 선택:**

#### A. 단일 게시물 (짧은 인사이트)
```
[Hook - 강렬한 첫 문장]

[핵심 메시지 2~3줄]

[CTA - 팔로우/공유 유도]
```

#### B. 쓰레드 (연결 게시물 3~5개)
```
🧵 1/5
[주제 소개 + Hook]

🧵 2/5
[핵심 인사이트 ①]

🧵 3/5
[핵심 인사이트 ②]

🧵 4/5
[실행 팁 / 명언]

🧵 5/5
[요약 + CTA]
```

### 4. 결과물 저장
```
output/threads/[날짜]_[주제]/
├── post.txt            ← 게시물 본문
└── source.txt          ← 원본 출처
```

## 쓰레드 작성 가이드

### 톤 선택

| 톤 | 특징 | 예시 |
|----|------|------|
| **~다/명령조** | 강렬하고 임팩트 있음 | "부자들은 시간을 산다." |
| **~습니다체** | 부드럽고 친근함 | "부자들은 시간을 삽니다." |

> **권장**: 쓰레드는 ~다/명령조가 더 효과적

### 글쓰기 규칙

1. **설명하는 느낌 ❌** - 팩트를 던지기
2. **명사로 끊기 ❌** - 완전한 문장으로
3. **리듬감** - 긴 문장 → 짧은 문장
4. **짧은 문장** - 한 문장 20자 이내 권장
5. **이모지 적절히** - 과하지 않게 (3~5개)

### 글자 수 제한
- **권장**: 300~500자
- **최대**: 500자 (쓰레드 제한)

### Hook 패턴 (첫 줄)

**효과적인 패턴:**
- 🔥 반전: "부자들은 절약하지 않습니다."
- 🔢 숫자: "30대에 은퇴한 사람들의 공통점 3가지"
- ❓ 질문: "왜 똑같이 일해도 월급만 다를까?"
- 💡 비밀: "억대 연봉자들이 절대 말 안 하는 것"

### CTA (Call to Action)

```
💾 저장해두고 리마인드 하세요
🔔 팔로우하면 매일 부의 인사이트
↩️ 공유해서 친구도 부자 마인드로
```

## 해시태그

**필수 (매 게시물):**
```
#richesse #리쉐스 #부자마인드
```

**추가 (3~5개):**
```
#돈공부 #재테크 #자기계발 #성공습관 #마인드셋
```
