# LinkedIn Post Generator

## 이 스킬은 언제 사용하나요?

- 사용자가 "링크드인 글 써줘" 요청 시
- 사용자가 "LinkedIn 포스트 만들어줘" 요청 시
- 사용자가 "전문가용 포스트 작성해줘" 요청 시

## 참조 문서

- `.agent/config/content_analyzer.md` - 콘텐츠 분석 프레임워크
- `.agent/config/brand_guide.md` - Richesse 브랜드 가이드

---

## LinkedIn 포스트 특성

| 항목 | 스펙 |
|------|------|
| 최대 글자 수 | 3,000자 (권장: 1,000~1,500자) |
| 이미지/카루셀 | 권장 (engagement 2~3배 ↑) |
| 해시태그 | 3~5개 권장 |
| 최적 게시 시간 | 화~목 08:00~10:00 |
| "더 보기" 클릭 전 | 첫 3줄만 보임 (Hook 중요!) |

---

## 실행 순서

### Step 1: 콘텐츠 수집

**유튜브 영상인 경우:**
```powershell
python -m yt_dlp --no-check-certificate --skip-download --write-auto-sub --sub-lang "en,ko" --convert-subs srt -o "output/source_video.%(ext)s" "<YOUTUBE_URL>"
```

**전사 파일 / 텍스트 파일인 경우:**
```powershell
Get-Content "<FILE_PATH>" -Encoding UTF8
```

**웹 아티클인 경우:**
read_url_content 도구 사용

### Step 2: 콘텐츠 분석

`content_analyzer.md`를 참고하여 추출:

- 핵심 메시지 1개
- 주요 인사이트 3~5개
- 실행 가능한 액션 2~3개
- 개인 경험과 연결할 수 있는 포인트

### Step 3: 포스트 구조 설계

**LinkedIn 포스트 구조:**

```
[Hook - 첫 3줄이 생명!]
.
.
.
[본문 - 스토리 + 인사이트]

[액션 아이템 / 교훈]

[CTA - 질문으로 마무리]

#해시태그1 #해시태그2 #해시태그3
```

> ⚠️ **중요**: 첫 3줄 이후에 점(.) 3개를 넣어 "더 보기"를 유도

### Step 4: 포스트 작성

**작성 규칙:**

1. **첫 3줄 = 승부**: "더 보기" 클릭 유도
2. **줄바꿈 많이**: 모바일 가독성 최적화
3. **스토리텔링**: 관찰 → 인사이트 → 교훈
4. **전문성 + 진정성**: LinkedIn 톤에 맞게

### Step 5: 결과물 저장

**저장 경로:**
```
output/linkedin/posts/[날짜]_[주제]/
├── post.txt            ← 포스트 본문
├── post_alt.txt        ← 대안 버전
└── source.txt          ← 원본 출처
```

## LinkedIn 포스트 작성 가이드

### ✅ Hook (첫 3줄) 패턴

| 패턴 | 예시 |
|------|------|
| **고백/반전** | "3번의 스타트업을 말아먹었다.\n\n4번째에 드디어 깨달은 것:" |
| **숫자 + 약속** | "유니콘을 만드는 4가지 조건.\n\n(저장해두세요)" |
| **질문** | "왜 '좋은 제품'이 항상 이기지 않을까?\n\nDavid Sacks의 답:" |
| **대담한 주장** | "대부분의 스타트업 조언은 틀렸다.\n\n특히 이것:" |

### ✅ 본문 구조

**방식 A: 리스트형**
```
1️⃣ 첫 번째 포인트
   → 부연 설명

2️⃣ 두 번째 포인트
   → 부연 설명

3️⃣ 세 번째 포인트
   → 부연 설명
```

**방식 B: 스토리형**
```
[상황/문제]

[전환점]

[해결/인사이트]

[교훈]
```

### ✅ CTA (마무리)

| 유형 | 예시 |
|------|------|
| **질문** | "여러분은 어떻게 생각하시나요?" |
| **경험 요청** | "비슷한 경험 있으시면 댓글로 공유해주세요" |
| **저장 유도** | "나중에 다시 보고 싶으면 저장 📌" |

### ✅ 해시태그

```
#스타트업 #창업 #기업가정신 #리더십 #비즈니스
```

LinkedIn에서는 해시태그를 포스트 맨 아래 배치

### ❌ 피해야 할 것

- 첫 줄에 링크 넣기 (engagement ↓)
- 줄바꿈 없이 길게 쓰기
- 지나친 자기 PR
- 추상적인 교훈만 나열

## LinkedIn vs X 차이

| 요소 | X (Twitter) | LinkedIn |
|------|-------------|----------|
| 글자 수 | 280자 | 1,000~1,500자 |
| 톤 | 캐주얼, 임팩트 | 전문적, 진정성 |
| 구조 | 짧고 강렬 | 스토리텔링 |
| 해시태그 | 2~3개 | 3~5개 |
| 이모지 | 최소 | 구조화에 활용 |
